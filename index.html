更有意義

Tocas UI 以更有意義的方式來命名類別名稱，

就像是你直接稱呼這個元素一樣地親切。

程式已經夠枯燥乏味了，別再讓人生也繼續枯燥下去。

HTML

<div class="ts buttons">
    <button class="ts button">卡莉絲</button>
    <button class="ts button">亞凡芽</button>
    <button class="ts warning button">橙希</button>
</div>

 

 

自訂顏色只需一行指令

在 Bootstrap 中，你會看到有人詢問如何更改顏色，

通常他們的答案是要求你另外寫一段 CSS 來覆蓋原生樣式，

在 Tocas UI 中，我們都幫你搞定了，甚至會自動替你的顏色配色。

 

更有趣的是，幾乎每一個 Tocas UI 的元素都支援自訂顏色。

SASS

.ts.button.sakura
       +button-theme(#FFB7C5, #7F5961)

HTML

<button class="ts fluid sakura button">羽田白音</button>
