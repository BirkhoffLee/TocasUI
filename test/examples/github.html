Tocas UI 2

Tocas UI 是基於 SASS 和 CSS3 的元件框架，可快速建立大型或小型網站的介面，

其用法與 Bootstrap 差異甚多，Tocas UI 以不雜亂，有意義為主。

可至：https://tocas-ui.com/ 觀看教學文件。

特色
更簡潔，沒有像 Bootstrap 那樣雜亂的樣式名稱。（如：.m-l-1, .p-x-2, .p-a-3）
以意義為樣式命名的主要精神。
以支援行動裝置為優先。
更加彈性的格線系統，並達到 16 格線。
據有回饋力的動畫。
元件之間可交互使用。
模塊不需要 jQuery（耶！）。
